# Resumen conciso

Cuando debas presentar el resultado de una tool o un texto largo al usuario:

1. Extrae solo la información relevante para la pregunta original.
2. Redacta un resumen de un máximo de tres frases, en español.
3. No repitas datos ya mostrados en turnos anteriores de la conversación.
